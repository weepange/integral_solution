# Integral Solver (Python)

Мощный консольный решатель интегралов с подробным выводом шагов решения, красивым интерактивным интерфейсом и поддержкой 5 различных типов интегралов.

## ✨ Основные возможности

- **Интерактивный CLI-интерфейс:** удобное меню, цветовое оформление, анимации ожидания.
- **5 типов интегралов:**
  1. Неопределённый `∫ f(x) dx`
  2. Определённый `∫[a, b] f(x) dx`
  3. Двойной `∬ f(x, y) dA`
  4. Криволинейный 1-го рода `∫_C f ds`
  5. Криволинейный 2-го рода `∫_C P dx + Q dy`
- **Подробные шаги решения:** для каждого интеграла выводится список примененных математических правил и подстановок (например, интегрирование по частям, линейность, табличные интегралы, формула Ньютона-Лейбница).
- **Продвинутый парсинг ввода:** полная поддержка неявного умножения (например, `2x`, `xcos(x)`, `(x-1)(x+1)`), автоматическое раскрытие и упрощение выражений.
- **Аналитическое и численное решение:** программа сначала пытается найти точное символьное решение (первообразную). Если аналитическое решение не найдено, автоматически применяется метод Симпсона для вычисления численного ответа.
- **Поддержка базовых функций:** `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `exp`, `log` (натуральный логарифм), `sqrt`, `abs`.

## 🚀 Как запустить

В проекте используется встроенный интерактивный режим. Достаточно запустить:

```bash
python3 integral_solver.py
```

Откроется главное меню, где можно выбрать тип интеграла и следовать инструкциям на экране.

## 📝 Примеры ввода

Движок поддерживает математический синтаксис, максимально приближенный к естественному:
- Сложение/вычитание: `x^2 + 3x - 1`
- Умножение: `2*x` или просто `2x`
- Возведение в степень: `x^2` или `x**2`
- Дроби: `1/x`
- Тригонометрия: `sin(2x)`, `cos(x)*exp(x)`

### Примеры использования в интерфейсе:
- **Определённый интеграл:** функция `sin(x)`, пределы `0` и `pi` (значения типа `pi`, `e` поддерживаются напрямую).
- **Криволинейный 2-го рода:** вычисление площади круга через криволинейный интеграл `∮ -y dx + x dy`.
  Ввод: функция P = `-y`, функция Q = `x`. Параметризация: `x(t) = cos(t)`, `y(t) = sin(t)`, пределы от `0` до `2*pi`. 

## 🛠 Технические детали
Решатель основан на самописном символьном движке (AST-деревья, правила дифференцирования и интегрирования), без использования тяжелых сторонних библиотек символьной математики вроде SymPy.
